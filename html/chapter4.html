  4. Suppose the equation \(Ax = 0\) has a solution \(x \neq 0\). Then for all rows \(i\):

    \[ \sum_{j} \alpha_{ij}x_j = \sum_{j}x_j(a^j, a^i) = (\sum_{j}x_j a^j, a^i) = 0 \]

    The vector \((\sum_{i}x_i a^i)\) is perpendicular to all \(a^i\), so it is the zero vector.

    proof, multiply each row by \(\overline{x_i}\) and sum all rows:

    \[ \begin{aligned} \overline{x_i}(\sum_{j}x_j a^j, a^i) &= (\sum_{j}x_j a^j, x_ia^i) = 0 \\ (\sum_{j}x_j a^j, \sum_{i}x_i a^i) &= 0 \\ \sum_{j}x_j a^j &= 0 \end{aligned} \]

    However since the \(a^i\) are linearly indenpendent, all \(x_i\) are zero. This is in contradiction with our assumption, so \(det A \neq 0\).
